Defuse AI — Reimagining Minesweeper with Dual AI

The 90s classic is now a cinematic bomb disposal simulator with two AI systems at work.

🧠 Dual-AI System
Colonel Rex (Gemini AI) delivers real-time tactical commentary on every tile reveal. An AI ON/OFF toggle lets players control API usage - OFF switches to instant fallback dialogue. 
Major Steele (Vertex AI) fires after each game, using function calling to query BigQuery mission archives and generate a data-grounded Strategic Debrief inside the overlay.

⚙️ Google Services Pipeline
Cloud Run hosts the Node.js backend. Secret Manager secures the Gemini key. BigQuery stores mission telemetry queried live by Vertex AI. Cloud Monitoring tracks requests. Firebase handles the leaderboard.

🔐 Security
Helmet CSP, rate limiting, ADC for Vertex AI and BigQuery, App Check on leaderboard writes.

The mines are buried. With Gemini, defusing them is cinematic. 💣
