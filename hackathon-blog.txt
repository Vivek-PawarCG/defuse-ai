Defuse - Reimagining Minesweeper with Gemini
The 90s classic is now a cinematic bomb disposal simulator with Gemini as a live "Field Commander."

🧠 AI Dynamic Metagame
Guided by Colonel Rex (powered by Google's Generative AI), the app analyzes real-time gameplay telemetry, tile coordinates and adjacent mines, to generate spoken tactical advice and panicked warnings on the fly.

*   Lifelines: Ask Gemini for intel on hidden tiles, risking accurate data or "signal interference."
*   Fail-Safe: API limits trigger a switch to local dialogue. Native Web Speech API ensures fast, tiny-bundle voice generation.
*  100% Accessibility, 100% Security with Testing & Best Practices

🚀 100% use of Google Services
*   Cloud Run: Serverless Node.js backend for UI and secure ML APIs.
*   Firebase: Global "Hall of Fame" leaderboard with real-time sync.
*   Cloud Logging: Enterprise-grade observability and real-time monitoring.
